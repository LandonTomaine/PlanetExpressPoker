# Orchestration Subagent Handoffs

## Backlog Review

- Source child PRD path
- Backlog path and diff summary
- Ask for missing PRD coverage, scope drift, slice size problems, dependency mistakes, and unclear handoff or validation
- Require findings only
- No file edits

## Slice Planning

- Source child PRD path
- Backlog path
- Selected `I*` slice and acceptance criteria
- Likely code areas if known
- Ask for a `plan-implementation-work` draft plan
- Ask for blocking ambiguity only
- Ask for exact validation
- Ask for a plan path recommendation under the capability `plans/` folder

## Changed-Code Review

- Source child PRD path
- Backlog path and selected `I*` slice
- Implementation plan path
- Changed files or diff summary
- Validation commands and results
- Tell the reviewer to use `review-changed-code`
- Require checklist-backed findings
- Avoid broad repo audit
