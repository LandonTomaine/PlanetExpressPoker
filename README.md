# Planet Express Poker

Lightweight planning poker with realtime rooms, persistent settings, animated reveal flow, and a chaotic Planet Express-inspired presentation layer.

## Stack

- `React`
- `TypeScript`
- `Vite`
- `Supabase`
- `Motion`
- `Tailwind CSS`
